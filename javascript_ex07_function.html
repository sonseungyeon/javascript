함수

함수를 선언하는 방법
  • function 함수명([parma1[,parma2[,...param]]]){내용}
    • 함수를 선언하면 함수가 호이스팅됨
    • 어디서도 사용이 가능하다
  • var x = function([parma1[,parma2[,...param]]]){내용}
    • 익명 함수
    • 변수 x생성된 시점 이후에 사용 가능하다
    • 변수 x로 함수를 호출한다
    • x(값1,값2...)
  • (function([parma1[,parma2[,...param]]]){내용})(값1,값2)
    • 익명 함수, 즉시 실행 함수
    • 함수의 선언과 실행을 동시에 한다.
