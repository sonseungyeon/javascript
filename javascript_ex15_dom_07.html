element 생성과 삭제

요소를 추가하는 방법
  1. innerHTML에 직접 문자열로 태그를 만들어 추가하는 방법
    • element.innerHTML = "<li>추가하는 아이</li>"
  2. 요소&노드 생성 함수를 이용하는 방법
    • 요소생성 : createElement("태그명")
    • 텍스트노드생성 : createTextNode("글자")
    • 노드추가 : appendChild(추가할 노드객체)
      • appendChild(추가할 노드객체)
      • insertBefore(추가할자식, 기존자식)
노드 삭제 방법
  1. 태그로 직접 지움
    • innerHTML=""
  2. 삭제 메서드 활용
    • 부모요소.removeChild(지울자식객체)
    • 지우려는아이.remove()
